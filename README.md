# Diwali-Sales-Analysis-using-Python

This project focuses on analyzing Diwali sales data to uncover customer purchasing behavior, sales trends, and key factors influencing revenue during the festive season. Using Python and data analysis libraries, the project provides meaningful business insights that can help organizations improve marketing strategies and customer targeting.

Objectives

Analyze sales performance during the Diwali season.

Understand customer demographics and buying patterns.

Identify top-performing product categories.

Provide actionable insights to boost festive sales.
